# Contributing to Screenie Client
If you're interested in contributing, then thanks in advance!  This project is maintained on a limited basis. I review and merge pull requests 
periodically, but response times may vary from days to weeks. Please be patient!

## How to Contribute
1. Fork the repository
2. Create a feature branch from `develop`
3. Make your changes
4. Submit a pull request to the `develop` branch
5. Wait for review (please be patient!)

## Pull Request Guidelines
- Keep PRs focused on a single feature or fix
- Update documentation as needed, including the README and docs if possible
- Follow existing code style and guidelines (see copilot-instructions.md file)
- Describe what your PR does and why

## Questions or Issues?
Open an issue on GitHub. I'll respond when I can.
